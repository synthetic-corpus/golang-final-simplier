A simple API

This is a simple API that has only five paths. The pats are CRUD fuctionality for a CRM systems.

All paths work with the same customer object

The Paths

  • POST /customers
  • GET /customers
  • GET /customers/:id
  • UPDATE /customers/:id
  • DELETE /customers/:id

JSON structure of a customer object

Key Data Type
ID string (auto generated)
Name string
Role string
Email string
Phone Number string
Contacted boolean
